# Jayant Web & AI Systems Portfolio

Welcome to the **Jayant Web & AI Systems** portfolio website. This site showcases our services, case studies, and the philosophy that drives our work.

## Mission

We aim to empower **10,000 Indian businesses** by adopting automated operational workflows that boost efficiency, reduce costs, and enable growth.

## Core Values

- **Ownership** – Full accountability for delivering flawless solutions.
- **Automation First** – Leveraging smart tools to eliminate manual bottlenecks.
- **Speed** – Rapid delivery without compromising quality.
- **Transparency** – Honest scopes, upfront pricing, and no surprise hourly bills.
- **Business Outcomes** – Aligning development with measurable goals (lead conversion, time saved, etc.).
- **Continuous Support** – Ongoing audits, updates, and performance monitoring.
- **Quality First** – Custom, robust platforms built for each unique workflow.

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

## Building for Production

```bash
npm run build
npm start
```

The site is optimized for performance with modern tooling and ready for deployment on Vercel or any static hosting platform.
